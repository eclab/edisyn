Korg SG Rack

Multi Patch Editor

This editor will work with the Korg SG Rack. It will not work with the Korg SG Pro X (or work well with it).

Communicating with Edisyn

Set your SG to have sysex enabled ("MIDI Exclusive: ENA").

Hints

  • Korg SG models come in roughly two versions: version numbers 1 to 7, and version 8. If you have version 8, you have more parameter options available. In each Timbre category, parameters available to version 8 only are marked with a [v8], parameters available to versions 1 to 7 only are marked with a [v1-7], and parameters available to both are not so marked. You tell Edisyn what version your machine is by marking the Version 8 checkbox.


Gotchas

  • The SG doesn't receive individual parameter changes: you'll have to make changes, then send or write the patch.
