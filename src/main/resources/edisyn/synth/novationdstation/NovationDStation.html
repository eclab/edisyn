Novation Drumstation / D Station

Single Patch Editor

This patch editor should work with any Drumstation with v1.3 firmware, and with the D Station. I cannot say if it will work right with earlier firmware than that (testing was done on a D Station).

Communicating with Edisyn

It'd be best if you selected PROGRAM mode. Set the Drumstation's MIDI Receive (Rx) channel to be the same as Edisyn's MIDI Send/Receive channel.

Sending and Writing a Patch to the Drumstation

The Drumstation has very, very limited sysex. Edisyn can send a single patch dump to the synth. If the Drumstation's "Write" switch is set to PROTECT, then when Edisyn issues a send, this sends the patch dump to the synth's current working memory as normal. If the "Write" switch is set to ENABLE, then Edisyn's send in fact overwrites the current patch slot.

If you have recently sent a patch to current working memory, you can also write that patch by switching the "Write" switch to ENABLE, selecting the desired patch (program) number, and pressing "Write". Remember that you can only write to slots 25...39; the earlier slots are read-only presets.

Receiving a Patch from the Drumstation

You have to send the patch manually from your unit. To do this, go to MODE 5 ("Save"). You should see an "SS" on-screen. This is for saving a single program. Then press the "+" (plus) button once. The Drumstation will send the patch to Edisyn, which should update itself in response.

Sending Individual Parameters

Edisyn can update a limited number of individual parameters via CC. See the "Controller Map" in your manual for a list of the parameters and their CC values: Edisyn will update them in real time. Other parameters can only be updated via patch dump.

Bank Sysex

The Drumstation can also send and receive Bank sysex messages: these are messages which contain all 15 writable patches (25...39). You can send this kind of message from the Drumstation to Edisyn by going to MODE 5 ("Save"), then selecting "SA", then pressing the "+" button once. Edisyn can save these messages, upload them to the Drumstation, or edit any one patch in them.

Gotchas

  • While it would be possible to program Edisyn to change the patch and then do a patch send, because we don't know if you've set the switch to PROTECT or to ENABLE, it seems foolish to do this. For this reason, Edisyn doesn't even display a patch number for the patch.

  • There are many myths surrounding the differences between the V1 Drumstation, V2 Drumstation, and D Station. But the truth is, if you have updated your Drumstation to firmware v1.3, then it is identical in all respects to the D Station, including polyphony and MIDI features.
