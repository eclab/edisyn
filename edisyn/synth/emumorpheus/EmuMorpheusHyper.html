E-Mu Morpheus and Ultra Proteus

Hyperpreset Patch Editor

This editor will work with the E-Mu Morpheus and the E-Mu Ultra Proteus.

Communicating with Edisyn

Set the MIDI ID to the same you'll be setting in Edisyn (typically 0). In your default MIDI Map, turn OFF Program Change Map, and turn ON Bank Select and Program Change. Feel free to reduce the Sysex packet delay to 0. Make certain that the Hyperpreset button is pressed.

Uploading a Proteus Patch

It's easy: just use Edisyn's E-Mu Proteus editor. Be sure to first turn ON the "Proteus Sysex" option in your unit.

Gotchas

  • Neither the Morpheus nor the Ultra Proteus can send to the current patch -- they can only write patches, or send individual parameters. But sending to the current patch is a critical function. Earlier versions of Edisyn used to write to patch 126, but as of Version 31, Edisyn just (slowly) sends individual parameters to the Morpheus/Ultra Proteus. This change is because it is unclear if the Morpheus/Ultra Proteus uses Flash RAM, and frequently writing to patch 126 will gradually burn it out.

  • You cannot request the current patch on the Morpheus or Ultra Proteus.
