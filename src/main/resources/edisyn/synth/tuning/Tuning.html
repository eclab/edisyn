MIDI Tuning Standard

Patch Editor

This is a tool designed to make create different tunings and scales and to upload them to and request them from synthesizers which support the MIDI Tuning Standard (or MTS). You can save your tunings as sysex patches.

Communicating with Edisyn

Synthesizers which adhere to MTS all have a device number. You must set the Synth ID in Edisyn's MIDI Devices window (see the "Change MIDI" menu) to be the same as your synth's MTS device number.

About the Tuning Dials 0 ... 127

MTS assigns a pitch to every MIDI note value from 0...127. Each of these knob pairs represents the frequency to assign to that value.

You compute the pitch by choosing an original note from the 12-tone-equal-temperment (A4-440hz standard modern western tuning) as well as an offset in cents from 0-100 above that note. Your resolution in cents is 100/216. The knob panel as it stands is a direct representation of the midi tuning standard sysex and allows you to manually input your desired values. However, this is not the recommended way to set these values. The Tuning menu option has utilities for calculating these knobs in bulk.

Tuning Utilities

The Tuning menu gives access to various utilities to make it easy to set different common tunings.

Load Scala File...

You can load tunings in the popular Scala format. These files end in ".scl" or ".SCL". Scala tunings are relative: you'll also need to specify what the root MIDI note and Root Frequency are.

EDO

EDO stands for Equal number of Divisions per Octave and is also commonly known as Equal Temperament. You'll also see the acronyms "ET" or "TET". Standard western tuning is EDO with A4 (MIDI note 69) set to 440hz, at 12 divisions per octave. Other tunings have their advantages and disadvantages and a decent summary of the topic can be found on Wikipedia. The calculate button gives the frequency of the selected midi number in standard western tuning, but feel free to use any positive number for frequncy for any midi note number as your root, all it is saying is "I want midi number x from my controller to equate to frequency y and to tune the rest of the notes using equal temperment with z divisons per octave."

Named Scales

These are scales that are either popular enough or notable enough to be named. You can find out more about them on Wikipedia. Many of these scales are standard scales found in the MTS presets on Yamaha 4-operator synthesizers such as the TX81Z. These are set up much in the same way that you set up EDO above.
