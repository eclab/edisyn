Korg Wavestation SR

Performance Editor

This editor is designed work with the Korg Wavestation SR. It will not work with any other Wavestation models (the original Wavestation, the Wavestation EX, or the Wavestation A/D).

Bank Sysex This patch editor knows about bank sysex messages (which group together multiple patches) as well as single-patch sysex messages. If Edisyn loads or receives a bank sysex message, you will be given the option to edit a patch from it, to save the whole bank sysex, or to upload the whole bank sysex.

Communicating with Edisyn Set your SR to receive sysex parameters, and also to receive program change and bank select commands, as well as All Notes Off. Make sure that the MIDI channel set for the SR matches the one you're using in Edisyn. You'll also want to make sure that you're at the Performance top-level menu, that is, both the "Perf" and "Edit" lights are lit, or else Edisyn may have trouble changing patches on request.

Gotchas

  • The Wavestation doesn't have an option to allow Edisyn to send parameters in bulk to its current working memory. But this is an important Edisyn feature, so Edisyn gives you two workaround options: via the Wavestation menu, you can either send parameters individually or in bulk. If you send parameters individually they will go to the current working memory, but it's slow to send them this way. If you send parameters in bulk, they will always go the first performance in the current bank.

  • Changes to parameters aren't displayed on the Wavestation unless that parameter is currently specifically selected on the Wavestation's screen.

  • Updating individual parameters is a little pokey. This is because Edisyn must insert a significant delay between successive parameter updates or they'll get dropped by the SR.

  • Changing either FX type takes a fair bit of time, because Edisyn is erasing every FX parameter. I'll try to speed that up in the future, but not now.

Notes

For the time being, I have elected to not have the patch editor update itself in response to manual parameter changes made directly on the synth. The SR is just too complex a beast.
