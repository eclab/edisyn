Red Sound DarkStar and DarkStar XP2

Single Patch Editor

This patch editor works on the DarkStar XP2 and (probably) the original DarkStar. It is not meant to work with the Vocoda.

Communicating with Edisyn

Set Protect to OFF (Utility 3). Set the Master MIDI Channel (Utility 8) to the same channel as Edisyn is using.

About the Menu

  • The Swap Parts... menu lets you, well, swap two parts with one another.

  • The Copy Part To... menu lets you copy a part to another part.


Hints

  • The DarkStar has the ability to emit a single part as sysex. Edisyn can receive this data and place it in any part of your choosing.


Gotchas

The DarkStar's sysex is extremely limited.

  • There is no way for Edisyn to request a patch. You will need to send a patch manually from the DarkStar via Save Program (Utility 5). Edisyn also responds to Save Part Voice (Utility 4) and Save All Programs (Utility 6 or 7).

  • Similarly there is no way to request a merge. To do a merge, request one as usual, then manually send a single patch from the machine.

  • The DarkStar cannot respond to individual patch parameter changes.

  • Weirdly, if you send all patches to Edisyn (Save All Programs, Utility 6 or 7), they're not in order. Instead, you'll get patches 01, 17, 33, ... etc., then 02, 18, 34, ..., then 03, 19, 35, ..., then 04, 20, 36, ...
