Waldorf M

Single Patch Editor

This editor is designed to work with at least Firmware 1.11.

Communicating with Edisyn
Put your M in Single Mode. Set Edisyn's MIDI channel to the same as the M's MIDI Channel. At present the M does not use its ID in its messages, though that may change. You may want Edisyn to update in real time to changes you make to knobs on the M. To do this, if you are connected over USB, set the M's "CControls Send" to USB or DIN+USB. If you are connected over DIN, set it to DIN or DIN+USB. Make sure that Global Bank Change and Program Change are turned on.

The Three, no Four, Modes
The Waldorf M can play in several different modes, and Edisyn changes what it displays depending on the mode chosen:

  • Classic Mode (similar to the Microwave 1)
  • Classic Mode with a Transient replacing the second Wavetable oscillator
  • Modern Mode (similar to the Microwave 2 / XT)
  • Algorithm Mode (both Wavetables are replaced with a variety of digital oscillator models)
These modes, and their respective options, are turned on by selecting the appropriate mode in the Oscillator Mode popup menu. If you select "Microwave 1 (Classic), you can then turn on the Transient option with the Enable Transient checkbox.

Transients and Transitions
In the past, the M called the PPG's Transient options "Transitions", and so did Edisyn. We've since switched back to calling them Transients.

About the Menu: Current Parts

When you update a parameter, if the M is in Multi Mode, it can update the parameter in one of its four parts. You can specify which part with Changing Parameters Updates Part .... Note that if you Send to Current Patch or Request Current patch in Multi Mode, nothing happens. (Obviously the Multi editor will work in this situation).

If the M is in Single Mode, updating parameters updates the Current Patch: individual parts are ignored.

Gotchas

  • You cannot load Transients in Edisyn, and Edisyn cannot display the filename of the Transient for a given number.

  • If you change a parameter in Edisyn, it will update the same parameter on the M. However when changing a parameter on the M, it will only update Edisyn if the parameter is a knob (that is, if it is mapped to a CC. See Page 84 of the manual). Other parameters are not emitted by the M.

  • For a very short while, the M had an additional Play Mode called "Mono Portamento". That has been rescinded, and Edisyn does not support it. Mono Portamento will be changed to some other Play Mode arbitrarily (probably "Polyphonic").

Hints
When you switch to a Waldorf M Editor window, it will automatically try to get the M to switch to Single Mode.

Microtuning Options
The Waldorf M supports microtonal tuning in Firmware 1.11. However it does not support the entire MTS tuning standard because it doesn't have enough firmware space. The M only responds to individual note tuning changes. That's okay! To dump all notes to the M, use Edisyn's Tuning patch editor, and check the checkbox labelled Send As Individual Parameters. This will cause Edisyn to send its dumps by sending each note separately to the M.
